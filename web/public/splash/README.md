# Splash Screens

This directory should contain splash screen images for iOS devices.

## Required Sizes

- iphone-se.png (640x1136) - iPhone SE, 5s, 5c, 5
- iphone-8.png (750x1334) - iPhone 8, 7, 6s, 6
- iphone-8-plus.png (1242x2208) - iPhone 8 Plus, 7 Plus, 6s Plus, 6 Plus
- iphone-x.png (1125x2436) - iPhone X, XS
- iphone-11-pro.png (1170x2532) - iPhone 11 Pro, 12 mini
- iphone-11-pro-max.png (1242x2688) - iPhone 11 Pro Max
- iphone-12-mini.png (1080x2340) - iPhone 12 mini
- iphone-12-pro.png (1170x2532) - iPhone 12 Pro
- iphone-12-pro-max.png (1284x2778) - iPhone 12 Pro Max
- ipad.png (1536x2048) - iPad, iPad mini
- ipad-pro.png (2048x2732) - iPad Pro

## Generating Splash Screens

You can use tools like:
- https://appsco.pe/developer/splash-screens
- https://www.appicon.co/#app-splash
- https://realfavicongenerator.net/

Or create them manually using the app's theme color (#1976d2) as background with the app logo centered.
