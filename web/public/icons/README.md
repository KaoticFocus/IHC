# App Icons

This directory should contain the following icon sizes for PWA support:

- icon-16x16.png (16x16)
- icon-32x32.png (32x32)
- icon-72x72.png (72x72)
- icon-96x96.png (96x96)
- icon-128x128.png (128x128)
- icon-144x144.png (144x144)
- icon-152x152.png (152x152)
- icon-192x192.png (192x192)
- icon-384x384.png (384x384)
- icon-512x512.png (512x512)

## Generating Icons

You can use an online tool like https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator to generate all required sizes from a single source image (recommended: 512x512 PNG).

## Icon Requirements

- Format: PNG
- Purpose: "any maskable" (for Android adaptive icons)
- Background: Transparent or solid color
- Recommended source size: 512x512px minimum
